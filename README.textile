= TOPPERS/SSP for STM32F4-discoveryテスト

TOPPERS/SSPをお試しでSTM32F4-discoveryで動作させてみました。

コンソールで使用するUARTはUSART1です。TxD=PB6、RxD=PB7を使用して下さい。

元はTOPPERS/SSPのcqarm用のツリーです。
http://www.toppers.jp/download.cgi/ssp_cq_starm_gcc-20120423.tar.gz

TOPPERSはリアルタイムOS μITRONの一種です。

その他詳しくは以下をご参照下さい。
